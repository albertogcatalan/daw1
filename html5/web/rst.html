Estructura
===============
				 
La **estructura** de la Web esta definida con las siguientes secciones: 
		
- **Inicio**: es la página principal de la Web donde se explica de forma breve la estructura de la misma.
		
- **Tablas**: contiene una tabla realizada de forma automática con Python basándose en los datos de las temperaturas de Zaragoza.
		
- **Formulario**: formulario donde insertar diferentes valores en los varios tipos de input posibles en un formulario HTML.
		
- **Documentación RST**: contiene esta documentación en formato RST para poder luego transformarla en HTML, PDF u otro tipo de formato de presentación de la información.
